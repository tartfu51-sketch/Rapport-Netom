# NETOM Terrain App

Application mobile web pour techniciens terrain – NETOM
Assistant de Bureau d'Études · Audit Énergétique & ETR

## Formulaires inclus
- ✅ Pré-chantier (état des lieux avant travaux)
- 🔜 Post-chantier
- 🔜 Dimensionnement chauffage
- 🔜 Dimensionnement groupe froid
- 🔜 Pré-audit ETR

## Déploiement sur Vercel (5 min)

1. Aller sur https://vercel.com
2. "Add New Project"
3. Importer ce dépôt GitHub
4. Framework : Create React App (détecté auto)
5. Cliquer "Deploy"
6. Ton URL : https://netom-terrain.vercel.app

## Installation sur mobile (PWA)

### iPhone (Safari)
1. Ouvrir l'URL dans Safari
2. Icône "Partager" → "Sur l'écran d'accueil"
3. NETOM apparaît comme une vraie appli

### Android (Chrome)
1. Ouvrir l'URL dans Chrome
2. Menu ⋮ → "Ajouter à l'écran d'accueil"
3. NETOM apparaît comme une vraie appli

## Développement local

```bash
npm install
npm start
```
